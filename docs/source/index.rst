유리카 커맨드 리스트
==========================

`SHUcreamDelight <https://discord.gg/wbeJHXf>`_ 메인 봇 유리카의 ``200530`` 현재 사용가능한 커맨드 목록.

오류가 있을시에는 @SHUcream#4484 호출
기재한 명령어 순서는 대충 많이 쓰이는 순서대로 정했음


짤 기능
-------
미리 등록해둔 키워드에 매칭되는 디씨콘이나 짤을 대신 출력하는 기능.

사용법:

* -*키워드*
* -랜덤 *키워드*

-키워드
++++++++
*키워드* 에 대응되는 짤이 있을 시 해당 짤 출력, 없을 시 무반응

-랜덤 키워드
++++++++++++++++
*키워드* 가 포함된 짤들이 존재할 시 그 중 하나를 랜덤으로 선택해 출력, 없을 시 무반응.


| 짤 등록은 별도의 ``=상점`` 커맨드를 통해서 등록 가능
| ※=로 시작되는 다른 명령어들과 달리, 짤 기능만은 중복문제를 피하기 위해 -로 시작하므로 유의할 것.
| ``200530`` 현재 등록되어있는 짤의 리스트를 확인하는 기능은 복구되지 않았음.

기억 기능
------------
메모 관리 기능 전반

=알려 키워드
++++++++++++
*키워드* 에 대응하는 기록이 있을 시 해당 메모 출력, 없을 시 에러 출력

=기억 키워드
++++++++++++++
| *키워드* 에 관련된 메모를 새로이 기록하는 기능.
| 만약 *키워드* 에 관련된 메모가 이미 존재할 시엔 ``=붙여`` 를 대신 사용해야함

=붙여 키워드
+++++++++++++
| 이미 존재하는 *키워드* 에 추가로 메모를 덧붙이는 기능.
| 만약 *키워드* 에 관련된 메모가 존재하지 않을 시엔 ``=기억`` 을 대신 사용해야함.

=찾아 키워드

=환율
-------------
| 환율 적용 후 얼마쯤 되는지 보여주는 기능.
| 환율은 은행과 국가마다 다르기에 실제와는 다소의 편차가 있을 수 있음

사용법
++++++
| =환율 액수 바꿀화폐 목표화폐
| 혹은
| =환율 액수 바꿀화폐 목표화폐 비교액수

**키워드**

* **액수** => 바꿀 액수. 예를 들어 $4라면 액수 파트엔 4 만 적어야함
* **바꿀화폐** => 원래 화폐 단위. 4달러를 원화로 바꾸고싶다면 *달러* 에 해당되는 부분.
* **목표화폐** => 최종적으로 알고싶은 기준 화폐 단위. 위의 예시에선 *원* 에 해당되는 부분.
* 비교액수(optional) => 결제시 화폐를 복수 선택할 수 있는 경우 어느쪽인지 이득인지 체크하는 용도.

**사용가능한 화폐 리스트**::

    원 위안 파운드 유로 달러 엔 홍콩달러 레알 캐나다달러 스위스프랑
    인도네시아루피 페소 필리핀페소 레우 크로네 포린트 쿠나 즈워티
    루피 링깃 크로나 루블 싱가포르달러 바트 리라 랜드

=주사위
--------------
| 혹은 =주사, 말 그대로 주사위 굴려주는 기능
| TRPG 용도의 주사위도 굴릴 수 있음

사용법
+++++++
=주사 면체 보정값

**키워드**

* | 면체(optional) => 굴릴 주사위의 면 수. 모든 면은 같은 확률을 가짐
  | TRPG식 다면체 주사위를 여러개 굴릴 경우 흔히 쓰는 dnd식 표기가능 e.g. 3d6
  | 만약 면체가 주어지지 않을 경우, 유리카는 1d6을 기본으로 굴림.

* 보정값(optional) => TRPG식 계산 한정으로 결과값에 합칠 보정값

=상점
---------

사용법
+++++++
=상점

1. 짤 추가 기능
+++++++++++++++++
| 처음에 소개한 짤 커맨드로 출력가능한 짤을 등록하는 기능
| 여태까지 등록한 짤의 수에 비례해서 등록 시의 소모 포인트가 증가함
| 증가식은 로그.

2. 짤 삭제 기능
+++++++++++++++++
| 본인이 등록한 짤만 삭제 가능.
| 등록 수에 관계 없이 고정 5pt 소모.

3. 숏소드 부여 기능
+++++++++++++++++++++
| 롤 변경용 숏소드 롤 체득
| 현재와선 별로 쓸 데 없음.
| 얻은 숏소드는 배포도 회수도 오케이.
| 다만 그걸로 마을사람등급(~마을사람)은 변경 금지

4. 마을 이주 기능
+++++++++++++++++
| 사실상 닉 색 변경 기능
| 고정 1천 포인트
| 자신 외의 다른 유저의 색을 변경하는 것도 가능함.

5. 마을 생성 기능
+++++++++++++++++++
| 신규 닉 색 생성 기능
| 색 선택 후 마을 명을 선정할 수 있음.
| 생성과 동시에 해당 색으로 이주되며, 이후 타인들도 이주가능하게 됨.
